ZOOM H2nの設定

2021-02-02

バイク車載動画撮影時のZOOM H2nの設定メモ

メニュー 設定値
ゲイン 2
録音モード MS STEREO S+6
LO CUT ON
COMP/LIMITER COMP1(GENERAL)
AUTO GAIN OFF
REC FORMAT WAV44.1kHz/16bit

この設定で、ウィンドスクリーンをつけてズボンの後ろポケットに入れる。 ＭＳ側マイクが外側に来るよう（＝ディスプレイが内側）な向きで。
